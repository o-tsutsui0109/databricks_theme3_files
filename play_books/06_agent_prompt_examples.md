# 研修用プロンプト例

## 目的

Agent Bricks / Knowledge Assistant の研修で、受講者がすぐに試せる質問例をまとめる。

---

# テーマ1: 問い合わせ一次対応・社内ナレッジ検索

## 基本質問

- 関西リテールから問い合わせが多いです。一次対応方針を教えてください。
- CSATが低い顧客への返信テンプレートを作ってください。
- 解約リスクHighの顧客に対して、サポート担当が確認すべきことを教えてください。
- API利用が多いEnterprise顧客から技術問い合わせが来た場合の確認事項を教えてください。
- 中部メディカルのような顧客に対して、どのようなエスカレーションが必要ですか。

## 期待する回答

- 顧客状況の要約
- リスク要因
- 一次回答案
- 内部確認事項
- エスカレーション先
- 次アクション

---

# テーマ2: 営業提案書・案件立ち上げ支援

## 基本質問

- 製造業向けに品質改善エージェントの提案書アウトラインを作ってください。
- 物流企業向けに納期遅延対策エージェントを提案したいです。提案メッセージを作ってください。
- SaaS企業向けの解約防止エージェントの類似案件カードを要約してください。
- 小売企業向けに販売・在庫改善エージェントのヒアリング項目を出してください。
- 顧客ヒアリングで確認すべき共通項目を10個出してください。

## 期待する回答

- 提案テーマ
- 顧客課題
- 解決策
- 期待効果
- ヒアリング項目
- 必要データ
- 次アクション

---

# テーマ3: 次アクション提案

## 解約防止

- 首都建設に対する次アクションを優先度付きで提案してください。
- 関西リテールは更新期限が近く、問い合わせも多いです。何をすべきですか。
- 解約リスクHighの顧客に対する共通アクションをまとめてください。

## 納期遅延

- O1008の納期遅延に対する次アクションを提案してください。
- 在庫不足による遅延が発生した場合の対応ルールを教えてください。
- 配送会社遅延と天候遅延で対応をどう変えるべきですか。

## 品質改善

- 千葉工場L02のモーターBで品質アラートが出ています。次に何をすべきですか。
- 夜勤帯で不良率が高い場合の確認事項を教えてください。
- 保全未実施と高温多湿が重なった場合の改善アクションを教えてください。

## 小売販売・在庫

- 新宿西口店のミネラルウォーターが欠品しています。次アクションを提案してください。
- 晴れの日に飲料販売が伸びている場合、店舗は何をすべきですか。
- キャンペーン中の商品で在庫が少ない場合の対応を教えてください。

---

# よいプロンプトの書き方

以下の要素を含めると、回答が安定する。

1. 対象顧客または対象レコード
2. 状況
3. 目的
4. 欲しい出力形式
5. 優先度や期限

## 例

```text
関西リテールについて、問い合わせ一次対応の方針を作ってください。
出力は、状況サマリー、リスク要因、顧客向け返信案、社内エスカレーション、次アクションの順にしてください。
```

```text
O1008の配送遅延について、顧客連絡文案と社内対応アクションを作ってください。
原因が未確定のものは断定しないでください。
```

```text
製造業の品質改善エージェントを提案するため、提案書のアウトラインとヒアリング項目を作ってください。
```
